Cryptocurrency Clustering

Unsupervised Machine Learning, KMeans, PCA, Python

Introduction

In this challenge, I used my knowledge of Python and unsupervised learning to predict if cryptocurrencies are affected by 24-hour or 7-day price changes. I started preparing the data using the StandardScaler() module from Scikit-Learn to normalize the data from the CSV file. Normalizing our data helps imporve our machine learning model, reducing the influence of outliers and reducing bias.

KMeans Clustering with Original Scaled Data

Using the elbow method, I created a scatter plot clustering the cryptocurrencies found in the dataset with 4 clusters. Looking at the graph to our left, we seem to have 2 outliers (clusters 1 and 2). While it's obvious that cluster 1 is an outlier, cluster 2 looks like it belongs to cluster 3. To uncover why this happens we will need to perform a principle component analysis (PCA).
